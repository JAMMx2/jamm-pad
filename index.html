JAMM Pad — Controls
Joint Art & Modeling Machine. Draw with a game controller, mouse + keyboard, or touch.

Game controller

  • Left stick move cursor
  • R2 / Cross (A) draw
  • Right stick up/down brush size
  • L1 / R1 change tool
  • Square (X) next color
  • Triangle (Y) glow on/off
  • Circle (B) undo
  • R3 sketch style
  • D-pad up/down symmetry
  • D-pad left/right mirror / rainbow
  • Options clear layer
  • Touchpad save PNG

Mouse + keyboard

  • Drag draw
  • Ctrl+Z / Ctrl+Y undo / redo
  • [ and ] brush size
  • Scroll zoom, Space-drag pan
  • Enter finish polygon
  • H this help

Touch

  • One finger draw
  • Two fingers pan + pinch zoom
  • Finish shape button for polygons
  • Open the Panel for layers, fill, projects
